# Арифматик операторууд
| Оператор  | Нэр  | Жишээ  |
|---|---|---|
| +  | Нэмэх  | 3+5=8  |
| -  | Хасах  | 8-3=5  |
| *  | Үржих  |  3*5=15 |
| /  | Хуваах  | 34/10=3.4  |
| %  | Үлдэгдэлтэй хуваах  | 34%10=4  |
| //  | Бүхэл хуваах  | 34//10=3  |
| **  | Зэрэгт  | 2**5=32  |


# Утга олгох операторууд
| Оператор  | Жишээ  | Тайлбар |
|---|---|---|
| =	| x = 5 |	x = 5	|
| =	| x = 5 |	| x = 5	|
| -=	 | x -= 3	| x = x - 3	|
| +=	 | x += 3	| x = x + 3	|
| *=	 | x *= 3	| x = x * 3	| 
| /=	 | x /= 3	| x = x / 3	| 
| %=	 | x %= 3	| x = x % 3	| 
| //= | 	x //= 3 | 	x = x // 3 |
| **= | 	x **= 3 | 	x = x ** 3 |
